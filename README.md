# 소스코드 
텍스트 사이에 소스코드를 삽입하려면 '의 기호를 사용. 

이 기호는 작은따옴표가 아니라 키보드에서 1왼쪽에 있는 ~과 같이있는 ` 흔히 백틱이라고 함. 

한 줄 짜리 소스코드는 `function add(x,y) {return x+y;}`

여러 줄 짜리 소스코드는 

```python

function add(x,y) {return x+y;]}

function add(x,y) {return x+y;}

add(x,y) {return x+y;]
```
